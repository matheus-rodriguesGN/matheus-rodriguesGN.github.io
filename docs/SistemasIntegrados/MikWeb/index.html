MikWeb

Dando sequência à nossa série sobre as integrações da Gerencianet, hoje vamos falar sobre o MiKWeb. Veja abaixo como realizar o procedimento:


Passos para a Integração#

  1. Acesse o menu API > Minhas Aplicações > Nova Aplicação;

  2. Informe o nome da integração, ATIVE o Modo de Compatibilidade (veja onde) e clique em Criar nova aplicação;

  3. Copie seu token de integração (veja onde) e acesse o portal MikWeb;

  4. No menu Configurações > Forma de Pagamento, escolha a opção Boleto Gerencianet ou Carnê Gerencianet no campo Forma de Pagamento;

  5. Em seguida, cole o token de integração no campo Chave de Integração;

  6. Para finalizar, clique no botão Salvar alterações.

Pronto! Sua integração já está funcionando.


Como dar baixa automática no MikWeb?#

Para que seja dada a baixa automática da cobrança em seu sistema MikWeb, é necessário cadastrar uma URL de callback na plataforma de emissão de boletos e carnês da Gerencianet. Da mesma forma, é preciso cadastrar uma URL de cancelamento para que o MiKWeb também realize uma baixa automática sempre que você fizer o cancelamento de uma cobrança na Gerencianet.

ATENÇÃO

Se o cancelamento for realizado no MiKWeb, não acontece a baixa automática na conta Gerencianet.


Passo a passo para cadastrar a URL de notificação:#

  1. Acesse o menu Integrações > Callback;

  2. No campo URL de notificação, informe o endereço de acesso de acordo com seu subdomínio, conforme modelo abaixo: http://demo.mikweb.com.br/confirmar_fatura_gn (no lugar da palavra demo informe seu subdomínio);

  3. Na caixa disponível ao lado, escolha a opção Cobrança Confirmada;

  4. Clique no botão Cadastrar;

  5. Faça o mesmo procedimento se desejar cadastrar uma URL de cancelamento.


Como verificar se a integração está funcionando?#

Você pode realizar testes para verificar se a URL está funcionando perfeitamente. Para isto, clique no botão Testar (veja onde). Ao clicar no botão, uma requisição é enviada para o MiKWeb e você recebe uma mensagem informando se o callback está funcionando ou não.


A integração falhou. Como verificar o erro?#

Se mesmo após a realização da integração não acontecer a baixa automática de pagamento ou se algum boleto não for emitido via integração, você pode verificar a origem do erro clicando no botão Logs, disponível ao lado da URL cadastrada (veja onde). Você verificará todas as requisições e a resposta de cada uma. Assim, analisando a resposta das requisições, será possível identificar o possível erro.


Ainda ficou com alguma dúvida? Envie um ticket para nossa equipe!
