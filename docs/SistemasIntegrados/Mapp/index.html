Mapp Sistemas

A Mapp Sistemas é uma empresa de tecnologia especializada em desenvolvimento de sistemas e aplicativos, com foco em empresas de transportes.


Veja como integrar a Gerencianet com a Mapp Sistemas#

Primeiramente, você deve criar uma aplicação na sua conta Gerencianet. O procedimento é muito simples:

  1. Acesse o menu API > Minhas Aplicações da sua conta Gerencianet;

  2. Clique no botão Nova Aplicação;

  3. Insira o nome que deseja utilizar e clique no botão Criar Nova Aplicação;

  4. Caso queira primeiramente testar as emissões, acesse a aba Desenvolvimento e salve o seu Client_Id e o Client_Secret

  5. Agora, caso já tenha testado e queira configurar o ambiente real (de produção), acesse a aba Produção e salve o seu Client_Id e o Client_Secret


Pronto! Com a aplicação na API criada, você deve acessar o Mapp Sistemas e enviar as credenciais criadas para contato@mappsistemas.com.br.
