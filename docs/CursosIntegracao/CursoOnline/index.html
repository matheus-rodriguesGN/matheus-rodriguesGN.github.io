Curso Online de Integrações

Pensando em oferecer novos meios de transmitir informações, a Gerencianet lançou o Curso EAD da Universidade Gerencianet, que são vídeos em formato de aulas com o objetivo de explicar, de maneira clara e objetiva, como utilizar nossas soluções em pagamentos online e integrar o seu sistema ou aplicação com a API Gerencianet.

O curso é gratuito, carga horária estimada de 6 horas e pode ser realizado de duas formas:

  1. Realizando o curso pela plataforma da Udemy. Assim, após realizar 100% do curso **iremos disponibilizar um certificado digital em seu nome** certificando-o da conclusão deste;
  2. Individualmente, assistindo somente as aulas que você desejar. Assim, os vídeos estão hospedados no YouTube e estão disponíveis na relação mais abaixo ou através da playlist.

A seguir, todas as aulas disponibilizadas individualmente no YouTube. Cabe frisar que o certificado digital é aplicável aos alunos que realizarem o curso pela plataforma Udemy .

  • Aula 01 - Primeiro intermediador de pagamentos do Brasil a emitir boletos registrados (Gerencianet)
  • Aula 02 - Conhecendo a Gerencianet
  • Aula 03 - Nossos produtos e serviços (Gerencianet)
  • Aula 04 - Visão geral sobre o Curso de Integrações Gerencianet
  • Aula 05 - API Gerencianet
  • Aula 06 - Métodos do Playground (ambiente de testes/sandbox da Gerencianet)
  • Aula 07 - Criando uma transação via Playground (ambiente de testes/sandbox da Gerencianet)
  • Aula 08 - Definindo método de pagamento via Boleto Bancário com a Gerencianet (por meio do Playground)
  • Aula 09 - Definindo método de pagamento via Cartão de Crédito com a Gerencianet (por meio do Playground)
  • Aula 10 - Implementando as notificações - Callback ou Retorno Automático (por meio do Playground)
  • Aula 11 - Instalando o Composer via Windows
  • Aula 12 - Instalando o Composer via Linux
  • Aula 13 - Criando uma transação por integração - Pagamento por Boleto Bancário (integração API Gerencianet)
  • Aula 14 - Criando uma transação por integração - Pagamento por Cartão de Crédito (integração API Gerencianet)
  • Aula 15 - Configurando sua URL de notificações (integração API Gerencianet)
  • Aula 16 - Visão geral sobre os Carnês (integração API Gerencianet)
  • Aula 17 - Criando uma transação por emissão de carnês (integração API Gerencianet)
  • Aula 18 - Visão Geral - Conhecendo o Marketplace (integração API Gerencianet)
  • Aula 19 - Criando transação por Marketplace (integração API Gerencianet)
  • Aula 20 - Obrigado por escolher a Gerencianet!
  • Aula 21 - Criando assinaturas (cobrança recorrente) pela API de integração Gerencianet

Se preferir, você pode acessar o curso de Integração com a API Gerencianet através da plataforma Udemy ou através de nossa playlist oficial no Youtube.
